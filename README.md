# Corneliuses

Corneliuses.com — a New Orleans-themed link hub built with Next.js (App
Router) and TypeScript, pointing to:

- [videos.corneliuses.com](https://videos.corneliuses.com)
- [brian.corneliuses.com](https://brian.corneliuses.com)
- [dinnerocracy.com](https://dinnerocracy.com)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4
- Mardi Gras-inspired palette, Playfair Display + Kaushan Script fonts
